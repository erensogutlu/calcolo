/* calcolo | Çok Fonksiyonlu Hesaplama Aracı */

Bu proje, modern HTML, CSS ve JavaScript kullanılarak geliştirilmiş, kullanıcı dostu ve mobil uyumlu (responsive) bir web uygulamasıdır. Uygulama en az 20+ farklı hesaplama fonksiyonunu destekler ve kullanıcıların çeşitli matematiksel işlemleri kolayca gerçekleştirmesine olanak tanır.

* Özellikler :

-> Mobil uyumlu (responsive) tasarım
-> Arama çubuğu ile hesaplama filtresi
-> 20+ farklı hesaplama özelliği :
  - Dört işlem (toplama, çıkarma, çarpma, bölme)
  - Karekök, üs alma, yüzde hesaplama
  - Vücut Kitle İndeksi, yaş hesaplama
  - Geometrik alan/hacim hesaplamaları (kare, daire, üçgen, dikdörtgen, küre)
  - Asal sayı kontrolü, faktöriyel, ortalama, mod alma
  - Ondalık → İkilik dönüşüm
  - Gün → Saat/Dakika dönüşüm
  - Faiz hesaplama, en büyük sayı bulma ve daha fazlası
  
* Kullanılan Teknolojiler : 

-> HTML / CSS
-> JavaScript

* Geliştirici : Eren Söğütlü

LİNK : https://erensogutlu-calcolo.netlify.app/

/* calcolo | Multifunctional Calculator */

This project is a user-friendly and mobile-friendly (responsive) web application developed using modern HTML, CSS and JavaScript. The application supports at least 20+ different calculation functions and allows users to easily perform various mathematical operations.

* Features :

-> Responsive design with mobile compatibility
-> Calculation filter with search bar
-> 20+ different calculation features :
  - Four operations (addition, subtraction, multiplication, division)
  - Square root, exponentiation, percentage calculation
  - Body Mass Index, age calculation
  - Geometric area/volume calculations (square, circle, triangle, rectangle, sphere)
  - Prime number check, factorial, average, mode
  - Decimal → Binary conversion
  - Day → Hour/Minute conversion
  - Interest calculation, finding the largest number and more

* Technologies Used :

-> HTML / CSS
-> JavaScript

* Developer : Eren Söğütlü

LINK : https://erensogutlu-calcolo.netlify.app/
